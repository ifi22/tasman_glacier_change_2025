# tasman_glacier_change_2025
Group project for GEOG208: Tasman Glacial retreat between 2005 and 2025 using Landsat 5-9 data. Completed on Google Colab using Python with data from Google Earth Engine.
Isla Fisher: Author of all data visualisations on the 'Forcing the Retreat' poster. This includes all of the colab notebook work, timelapse, CIR images, glacier ice volume graph, and DEM. Minor editing role of the 'Forcing the Retreat' poster and provided information about the visualisations produced for Figure captions.
